HTML IMAGES

  • HTML img tag is used to display image on the web page. HTML img tag is an empty tag that contains attributes only, closing tags are not used in HTML image element.
  • img tag have some attributes are use for display image on web page.
  • The src attribute, src stands for "source", that is path of image URL.It is a necessary attribute that describes the source or path of the image. It instructs the browser where to look for the image on the server.
  • Alt Attribute used to define an "alternate text" for an image. The alt attribute defines an alternate text for the image, if it can't be displayed. The value of the alt attribute describe the image in words. The alt attribute is considered good for SEO prospective.
  • Width and Height specifies the size of image to display on webpage.
